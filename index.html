How To Add CSS in HTML

There are three way to add css

  1. External
    2. <head> <link rel="stylesheet" href="./css/style.css"> </head>
  2. Internal
    3. <head> <style> #code1{ color: blue; } </style> </head>
  3. Inline
    4. <p style="color: crimson">This we will write in inside tag itself.</p>

    NOTE: Inline css get most priority unless and until !important is written for the same selector somewhere else.

    NOTE: Internal CSS & External CSS priority is depends on sequence i.e whichever will be written at last will be applied.

CSS Selectors

For styling you have to select element and that is called selector.

There are mainly three type of selectors:-

  1. Element Selector
    2. p{ color: goldenrod; }
  2. ID Selector

    3. Id is unique and can be used only one time per id.

    #id-selector { color: red; }
  3. Class Selector

    4. There can be several class and can be used multiple times.

    .class-selector { color: seagreen; }

CSS Colors

In css color can be specified using

  • Predefined Color Name

    • Modern browser support 140 named colors

  • RGB
    • rgb(red, green, blue)
    • Color value between 0 to 255
    • Black : rgb(0, 0, 0)
    • White : rgb(255, 255, 255)
    • Red : rgb(255, 0, 0)
    • Green : rgb(0, 255, 0)
    • blue : rgb(0, 0, 255)
  • RGBA
    • rgba(red, green, blue, alpha)
    • alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not fully transparent at all)
    • RGBA(255, 99, 71, 0.3)
  • HEX
    • #rrggbb
    • rr (Red), gg (Green), bb (Black)
    • Hexadecimal values between 00 and ff (same as decimal 0-255)
    • Red: #ff0000
    • Black: #000000
    • White: #ffffff
  • HSL
    • hsl(hue, saturation, lightness)
    • hue, saturation and lightness (HSL)
    • Hue is the degree on the color wheel from 0 to 360. 0 is red, 120 is green, and 240 is blue.
    • saturation is percentage value, 0% means shade of gray, and 100% is the full color.
    • lightness is also a percentage, 0% is black, 50% is neither light or dark, 100% is white.
    • Light Orange: HSL(0, 100%, 50%)
  • HSLA
    • hsla(hue, saturation, lightness, alpha)
    • alpha parameter is a number between 0.0 (fully transparent) and 1.0 (not fully transparent at all)
    • HSLA(0, 100%, 50%, 0.3)

CSS Backgrounds

Background color changed

  • If your container size is big and your image is small, than your image will repeat itself to fill the container. You can use property called background-repeat: no-repeat;

  • You can also position your image along X and Y axis. Use background-position: bottom center;

  • You can also set background sizes along X and Y axis.i.e background-size: 500px 500px;

    You can also give size in percentages, but when you do so then percentage will be calculated according to your container's height and width. background-size: 50% 50%;

    • Contain
      • This do not cut your image.
      • This don't distort your image.
      • This might not fill your container.
      • You have to maintain the ratio of containing div.
    • Cover
      • It automatically adjust size to 100x100
      • Due to automatic adjustment it cuts image automatically to fill background.
      • This vary from browser to browser how it will be displayed.
      • This generally used in patterns.
  • Background is still and content in moving, to achieve this you should use this property i.e background-attachment: fixed;

  • Note:
    • When you give size (100% by 100%) then this disturb your ratio, your image may look stretched.
    • Cover don't stretch your image but while trying to fill the container it cuts image. Use with patterns.
    • Use contain when your size and ratio of div and image are same. ratio plays very vital role when you use contain. Ratio of div and image should be same.

  • You can use all background property at once. Here is the sorthand:
    • background: background-color background-image background-repeat background-attachment background-position ;
      • Example: background: #00ff00 url('/images/banner.jpg') no-repeat fixed center;

  • Image Sprites is made up of small images into one big image and again in css that will be used as individual image.
  • Image Sprites only be used as background image.

Very Important Topics in CSS

CSS Display and visibility

  • Every elements in html is rectangular box. How that rectungular box will behave ? it depends on its display property.
  • Every elements have its own default property which is provided by browser. It may vary from browser to browser.
  • Mainly there are four property which you will generally use:
    1. Block
      1. It takes full width.
      2. Always starts with new line.
      3. Width and Height can be applied.
    2. Inline
      1. It takes width as much as required according to content.
      2. It don't starts with new line.
      3. Width and Height can not be applied.
    3. Inline Block
      1. It takes width as much as required according to content.
      2. It do not starts with new line.
      3. Width and Height are applicable.
    4. None
      1. It just removes the block.
  • Block and Inline are default.
  • There are two way to hide element
    1. Display: none
      • when we apply this then it leaves place means completely removed from html.
    2. visibility: hidden
      • when we apply this then it not leaves place, means it occupy its place.

CSS Position

  • It defines element's position in website.
  • There are four type of position.
    1. Static
      • This is default.
    2. Relative
      • It moves with respect to its own position.
      • You have supply value to it.
      • If you have given position: relative; to div or element then nothing will happen until you specify direction.
      • After specifying position element will move but it will not leave its place. Means no other element can occupy that place.
        • Positoned to it's own position

    3. Fixed
      • This is according to viewport.
        • Fixed Positioned
    4. Absolute
      • It start it's calculation from positioned parent.
      • Positioned parent are those parent which contains position properties other than static.
      • If you have given position: absolute; to div or element then it imidiately leave its position.
      • After leaving its positioned other element will occupy that weather you provide direction or not i.e top, bottom, left & right.
        • Positoned a/c to it's direct positoned parent
  • These four properties are applied only when your position is relative, fixed or absolute.i.e
    1. Top
    2. Bottom
    3. left
    4. right

CSS Layers and z-index

  • Z-index only works for positioned element.
  • Elements with position absolute, relative or fixed can only eligible for z-index.
  • Higher z-index element will be on top.
  • z-index will arrange itself in decreasing order.

CSS Float and Clear

  • Float
    • This is used to move element left and right.
    • Whenever we use float li comes out of ul. Means li comes out from its container. To handle this problem use overflow: auto;
  • Clear
    • It tells that weather any element can be placed in right or left.
    • Clear can be applied on floated element as well as non floated element.

  • This was introduced in css 3
  • If we write max-width: 768px it means that devices greater than 768px will not render the changes.
  • max-width: Style will be applied less than given 400px
  • min-width: Style will be applied above given 400px
  • @media screen and (max-width: 768px) { #res-menu-header { width: 50%; } #res-logo-header{ width: 50%; } } @media screen and (max-width: 400px) { #res-menu-header { width: 100%; } #res-logo-header{ width: 100%; } }

    • Whichever element have this attribute and value will be selected. a[href="https://www.example.com"] { color: red; }

CSS Flexbox

  • Responsive website without using float and position
  • flex-direction: row-reverse: This will shift content to right direction in reverse order.
  • flex-direction: column-reverse : This will arrange item in vertical direction in reverse order.
  • flex-wrap: nowrap;: This will try to fit all child into parent by adjusting their width.It will ignore width given to the child.
  • flex-wrap: wrap;: This will consider the width given to the child. If any extra child will be there then that will be shifted into next line.
  • flex-flow: row wrap-reverse: This is shortcut for giving direction and wrap/nowrap.

    • Difficult to deal with normal css

  • justify-content: flex-start: It works horizontally. This is by default.
  • justify-content: flex-end: This will shift all child to the right side.
  • justify-content: center: This will shift all child to the center.
  • justify-content: space-around: This will give equal space around all child elements.
  • justify-content: space-between: This will give equal space only between child elements. It will not take any space from it's right or left.
  • align-items: stretch/center/flex-start/flex-end: It works vertically. It is very usefull when you have given excess height to parent then all child height will streach by default. So for controling that you have to use this property.
  • You can also change order of child element. Just select all that element and give order: 4 like this.
  • flex-grow: ratio: This will allow the child to occupy extra spaces according to ratio.
  • flex-shrink: ratio: This will allow you to shrink specific elemnt if some extra element are added to the group.
  • flex-basis: 300px: It tells that what is the initial width.
  • align-self: center/flex-start/flex-end: This is styling individual child element.

CSS Grid

  • When we apply display: grid: Then nothing will happen, it just divide things in row.
  • display: inline-grid: This will take space according to the content.
  • Create column in grid with custom values by using grid-template-columns: auto 50% 100px;
  • Increase height of rows grid-template-rows: 100px 200px auto;
  • grid-row-gap: 20px;
  • grid-column-gap: 30px;
  • To move grid justify-content: end;

    • Grid items properties

  • To merge columns
    • grid-column-start: 1;
    • grid-column-end: 3;
    • grid-column: 2/4;
  • To merge rows
    • grid-rows-start: 1;
    • grid-rows-end: 3;
    • grid-row: 2/4;
  • To merge rows and column both at once
    • grid-rows-start: 1;
    • grid-rows-end: 3;
    • grid-row: 2/4;
